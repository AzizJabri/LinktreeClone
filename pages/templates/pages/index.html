WELCOME TO LINKTREE CLONE

This is a linktree clone created by @AzizJb using Django and Postgresql for the backend, HTML and Bootstrap for the frontend.

This project is just for learning purposes.

FEATURES

Authentication Linktree Links Dashboard

You can login, register, modify your password and reset it.

Click here to login.
Click here to register.

You can create a linktree with a custom slug for the url (for now the pages are not customizable).

you can access it by visiting {{ request.get_host }}/your_custom_slug.

Note: your linktree slug is case sensitive!

You can add, update and/or delete links to the chosen linktree page

Note: the link url must be a valid URL (eg https://www.sitename.com)

Your dashboard show usefull informations about your profile and your linktrees

  • Last login date
  • Number of linktrees
  • Number of links in every linktree
  • (adding more features in the near future)
